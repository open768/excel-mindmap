# excel-mindmap
a portable mindmap app - for locked down environments

So you've found yourself in an environment thats locked down. you're not allowed to install even the most basic productivity tools that you have come to rely upon.

excel-mindmap is an experiment to find out just whats possible using just vba in excel to give you some sense of sanity. its the first app in the experiment. 

a pre-requisite is you'll need to be able to enable macros, some places even lock that down !
